# Grocino Website 🛒

Grocino Website is a modern and responsive grocery shopping application built with React.js. It provides users with a seamless online shopping experience through an intuitive interface, allowing them to browse products, explore categories, manage their shopping cart, and enjoy a clean, responsive design.

## 🚀 Features

* Product Listing
* Category Filtering
* Shopping Cart
* Product Search
* Responsive Design
* Reusable React Components
* Fast Navigation
* User-Friendly Interface

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* Vite

## ▶️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/Grocino-Website.git
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open the application in your browser.
